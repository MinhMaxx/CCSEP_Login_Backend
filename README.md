# server

This is the NoSQL exploitable backend service
Please switch to other branch for the NoSQL patches

## Project requirement
MongoDB
```
sudo apt update
sudo apt-get install -y mongodb-org
```
NodeJS + npm
```
sudo apt update
sudo apt install nodejs npm
```


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
nodemon
```
